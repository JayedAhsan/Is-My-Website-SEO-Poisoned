Is My Website SEO Poisoned?

Compare direct visits, Google clicks, and Googlebot responses.

A simple page to inspect how a URL behaves across three request types. If the returned page shifts between those views, it can be a sign of cloaking, poisoned SEO landing pages, or suspicious redirect behavior.

What it checks

Direct visit

Requests the target like a normal visitor.

Google search click

Adds a Google referrer to imitate search traffic.

Googlebot view

Uses a Googlebot user agent for crawler-facing content.

Run a check

Enter a URL and inspect what changes between the three request profiles.

Awaiting analysis

Run a check to compare the three responses.
